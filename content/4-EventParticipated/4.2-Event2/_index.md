# EVENT REPORT: AWS COMMUNITY MEETUP (06/06/2026)

### Event Objectives
*   Explore advanced Cloud computing technologies, focusing on security, containerization, and database management.
*   Understand the integration between Machine Learning and Generative AI (GraphRAG) with AWS infrastructure.
*   Approach real-time cloud architectures for gaming and interactive applications.
*   Develop soft skills for effective teamwork and map out a clear career transition path to Cloud and DevOps positions.

### Key Highlights

#### 1. Combining AWS WAF with Machine Learning
*   **Cybersecurity:** Discover how traditional rule-based Web Application Firewalls (WAF) can be enhanced with Machine Learning to detect and mitigate sophisticated, zero-day cyber attacks on the AWS environment.

#### 2. Docker - Containerization Technology
*   **Core concepts:** Clarify the differences between containerization and virtualization. Discuss how Docker packages applications and their dependencies into standardized units for software development, ensuring consistency across different computing environments.

#### 3. Multiplayer on the Cloud: Godot & AWS WebSockets
*   **Real-time architecture:** Present how to build scalable multiplayer architectures by connecting Godot game engine clients with AWS WebSockets, highlighting low-latency, bidirectional communication in the cloud.

#### 4. AWS Neptune and GraphRAG
*   **Advanced databases:** Introduce AWS Neptune as a fully managed graph database service. Explain its crucial role in building a Graph Knowledge Base to power GraphRAG, significantly improving the context and accuracy of Generative AI applications.

#### 5. Career Journey & Effective Teamwork
*   **Professional development:** Share a practical self-study roadmap transitioning from an IT Helpdesk role to Senior Sysadmin and eventually to Cloud/DevOps. Accompanied by practical strategies to improve communication, task delegation, and effective collaboration in tech teams.

---

### What I Learned

#### Design Thinking
*   **Container-First Mindset:** Shift from traditional deployment models to containerized environments to ensure application portability, completely eliminating the "it works on my machine but fails on the server" issue.
*   **Contextual Data Modeling:** Understand how graph databases (AWS Neptune) map complex data relationships more effectively than traditional relational databases for specific AI problems.

#### Technical Architecture
*   **Security Automation:** Grasp the architecture of integrating ML models with AWS WAF to create an intelligent, auto-updating line of defense that protects web applications from risks.
*   **Event-Driven & Real-time:** Understand how to deploy AWS WebSockets to maintain persistent connections, which is extremely crucial for real-time features and synchronization.

---

### Application to Work & Project
*   **Backend Security Enhancement:** Apply the concepts of AWS WAF discussed in the event to establish a solid security layer for the EC2 instances and Amazon RDS database of the Football Field Booking Website project, proactively filtering malicious web traffic.
*   **Containerized Deployment:** Use Docker to package the Java Spring Boot backend. This process will streamline the deployment pipeline, ensuring the application runs consistently whether being tested in a local environment or auto-scaling on AWS.
*   **Career Development & Collaboration Skills:** Apply Agile/Scrum teamwork strategies and the advancement roadmap to strengthen engagement with project members. These perspectives help shape problem-solving thinking and hone core competencies, creating a perfect stepping stone for the upcoming Backend Developer graduation internship.
