This page summarizes the entire activity log (Worklog) during the **Workforce Bootcamp - First Cloud AI Journey** internship program.

The internship program was completed in **12 weeks** (from April 20, 2026, to July 12, 2026). Throughout this period, the workflow was divided into two main phases:
*   **First 8 weeks:** Focused on researching foundational cloud computing theory and hands-on practice (Labs) with the core services of the AWS ecosystem (Compute, Storage, Database, Security, Networking).
*   **Last 4 weeks:** Applied the acquired knowledge into a real-world project, covering architectural design, writing source code (Java Spring Boot MVC), and successfully deploying the "Football Field Booking Management System" onto a real AWS environment.

Below is the detailed list of tasks performed each week:

**Week 1:** [Introduction to the training roadmap, creating an AWS account, and drafting the architectural diagram](1.1-week1/)

**Week 2:** [Researching and practicing basic Compute and Storage services: IAM, EC2, EBS](1.2-week2/)

**Week 3:** [Exploring Amazon VPC virtual networks, managing AWS Budgets, and the AWS Support system](1.3-week3/)

**Week 4:** [Building Custom VPC infrastructure, configuring multi-layer network security, and EICE](1.4-week4/)

**Week 5:** [Scaling compute with EC2 Auto Scaling, practicing AWS Backup and Storage Gateway](1.5-week5/)

**Week 6:** [Hosting static websites on Amazon S3, CDN acceleration with CloudFront, and data migration](1.6-week6/)

**Week 7:** [Identity management (Cognito), security assessment (Security Hub), and automation (Lambda)](1.7-week7/)

**Week 8:** [Exploring the Database ecosystem, deploying Amazon RDS, and migrating data with DMS](1.8-week8/)

**Week 9:** [Brainstorming, requirement analysis, and AWS architecture design for the Football Field Booking system](1.9-week9/)

**Week 10:** [Developing and packaging the Spring Boot MVC backend source code for the project](1.10-week10/)

**Week 11:** [Deploying the web system onto the AWS platform (EC2, RDS, S3) and configuring a Custom Domain](1.11-week11/)

**Week 12:** [Project handover, finalizing the Internship Report (Workshop) page, and administrative procedures](1.12-week12/)
